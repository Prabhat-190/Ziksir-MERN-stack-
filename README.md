# Ziksir MERN Notes App

Author: Prabhat Kumar (IIT Kharagpur)

A simple MERN stack notes application with JWT authentication.

## Features
- User registration & login
- JWT-based authentication
- Create & delete personal notes
- User-specific data isolation

## Tech Stack
- MongoDB
- Express
- React
- Node.js

## Run Instructions
Backend:
npm install
npm start

Frontend:
npm install
npm start
